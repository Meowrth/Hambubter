
# 🍔 Hambubter ($HBUB) - Too Hungry to Stop

Hambubter는 햄스터, 버터, 빵이 만나 탄생한 전설적인 밈코인입니다.

전설에 따르면 Hambubter 종족은 **버터 부족 사태** 이후 블록체인 위에 새로운 굽기의 왕국을 세웠습니다.
